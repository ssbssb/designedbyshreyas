📝 How to Customize

1. Update Your Information

Edit these files with a text editor (Notepad, VS Code, etc.):

  • Name: Change "SHREYA SHAKE" to your name (in index.html)
  • Bio: Update the hero section text
  • Experience: Edit the 4 timeline cards
  • Email: Update contact information
  • Social Links: Add your real social media URLs

2. Change Images

Replace these image URLs with your own (or use Unsplash links):

  • Profile image (Hero section)
  • Design philosophy image (About section)
  • Project images (Exhibits section)

3. Modify Colors

Edit these CSS variables in styles.css:

--primary-bg: #0a0e27;
--highlight-yellow: #d4ff00;
--highlight-cyan: #00d4ff;

💻 Development Setup (Optional)

Using VS Code + Live Server (Recommended)

1. Download VS Code
2. Install "Live Server" extension (by Ritwick Dey)
3. Right-click index.html → "Open with Live Server"
4. Your site auto-refreshes as you edit files

Using Python (Alternative)

# Go to folder in terminal, then run:
python -m http.server 8000

Then visit: http://localhost:8000

🌐 Deploy to Web

Option 1: Netlify (Easiest - Free)

  1. Push folder to GitHub
  2. Go to Netlify.com
  3. Connect your GitHub repo
  4. Deploy (instant publishing!)

Option 2: Vercel

  1. Push to GitHub
  2. Go to Vercel.com
  3. Import project
  4. Auto-deploys on each push

Option 3: GitHub Pages (Free)

  1. Push to GitHub repo
  2. Settings → Pages
  3. Select "main" branch
  4. Your site goes live automatically
🎉 Your portfolio will have a live URL and be accessible worldwide!

📧 Make Contact Form Work

The form currently shows demo notifications. To send real emails:

Option A: Formspree (Easiest)

  1. Go to formspree.io
  2. Create account & get form ID
  3. In index.html, find the form tag
  4. Add: action="https://formspree.io/f/YOUR_ID"
  5. Done! Emails will now be sent to you

Option B: Netlify Forms (If using Netlify)

Just add netlify attribute to form - automatic!
